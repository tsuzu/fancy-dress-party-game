# 🎵 🎃 仮装狂騒曲当てゲーム 🎵

YouTubeに投稿されている音声を聴いて、誰が歌っているかを当てるクイズゲームです。

## 🎮 ゲームの特徴

- YouTubeの動画を埋め込んで音声のみを再生
- 映像は完全に隠されており、声だけが頼り
- 4択クイズ形式で直感的に遊べる
- レスポンシブデザインでPC・タブレット・スマホに対応
- スコア表示と正解率の計算

## 🚀 使い方

### 1. ローカルサーバーで起動

ブラウザのCORS制限により、ローカルファイルとして直接開くことはできません。
以下のいずれかの方法でローカルサーバーを起動してください。

#### Python 3を使用する場合
```bash
python -m http.server 8000
```

#### Python 2を使用する場合
```bash
python -m SimpleHTTPServer 8000
```

#### Node.jsのhttp-serverを使用する場合
```bash
npx http-server -p 8000
```

#### VS Codeの拡張機能を使用する場合
Live Serverなどの拡張機能をインストールして、右クリックから「Open with Live Server」を選択

### 2. ブラウザでアクセス

サーバーを起動したら、ブラウザで以下にアクセス：
```
http://localhost:8000
```

## 🎯 問題の追加・編集方法

`data/questions.json`ファイルを編集することで、問題を追加・変更できます。

### 問題データの形式

```json
{
  "questions": [
    {
      "id": 1,
      "videoId": "YouTube動画ID（11文字）",
      "startTime": 0,
      "question": "この曲を歌っているのは誰？",
      "options": [
        "選択肢1",
        "選択肢2",
        "選択肢3",
        "選択肢4"
      ],
      "correctAnswerIndex": 0,
      "explanation": "正解の説明やトリビア",
      "songTitle": "曲名",
      "originalArtist": "原曲アーティスト"
    }
  ]
}
```

### YouTube動画IDの取得方法

YouTube動画のURLから11文字のIDを取得します。

例：
- URL: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- 動画ID: `dQw4w9WgXcQ`

## 🎨 カスタマイズ

### カラースキームの変更

`css/style.css`の`:root`セクションでカラーを変更できます：

```css
:root {
    --primary-color: #FF6B6B;      /* メインカラー */
    --secondary-color: #4ECDC4;    /* アクセントカラー */
    --background-color: #1A1A2E;   /* 背景色 */
    --text-color: #EAEAEA;         /* テキスト色 */
    --correct-color: #51CF66;      /* 正解時の色 */
    --incorrect-color: #FF6B6B;    /* 不正解時の色 */
}
```

## 💡 注意事項

1. **インターネット接続が必要**：YouTube APIを使用するため、オンライン環境が必要です
2. **動画の埋め込み許可**：使用する動画が埋め込み許可されている必要があります
3. **著作権**：使用する動画は著作権に配慮して選定してください
4. **ブラウザの自動再生ポリシー**：一部のブラウザでは自動再生がブロックされる場合があります

## 🔧 トラブルシューティング

### 問題データが読み込まれない
- ローカルサーバーを起動しているか確認してください
- `data/questions.json`ファイルが正しい場所に存在するか確認してください
- JSONの構文エラーがないか確認してください

### 動画が再生されない
- インターネット接続を確認してください
- YouTube動画IDが正しいか確認してください
- 動画が削除されていないか、埋め込み許可されているか確認してください

### 音声が聞こえない
- ブラウザやデバイスの音量設定を確認してください
- ブラウザの自動再生がブロックされている場合は、再生ボタンをクリックしてください
